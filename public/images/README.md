# Image Placeholders

This folder should contain the following images:

## Root images (public/images/)
- `hero-bg.jpg` - Concert stage background for hero section (recommended: 1920x1080px)
- `band-group.jpg` - Band group photo for biography section (recommended: 1600x900px)
- `events-bg.webm` - Animated video background for events section (WebM format, will autoplay looped and muted)

## Member photos (public/images/members/)
- `matteo.jpg` - Matteo Ballarin (400x400px square)
- `andrea.jpg` - Andrea Ghion (400x400px square)
- `denis.jpg` - Denis Ronchese (400x400px square)
- `manuel.jpg` - Manuel Smaniotto (400x400px square)

All member photos will be displayed as circular, so make sure the subject is centered.
